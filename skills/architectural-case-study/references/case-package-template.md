# Case Package Markdown Template

Use this template for `case.md`. Keep it concise enough to become a static case-library page.

```markdown
# <中文标题或项目名> / <English Project Name>

## 快速信息

- 项目名称：
- 建筑师 / 事务所：
- 地点：
- 年份：
- 类型 / 功能：
- 状态：
- 主要来源：

## 一句话概括

用 1 句话说明这个案例为什么值得研究。

## 项目背景

用 1-2 段说明委托背景、城市/场地语境、项目任务或展览/机构背景。每段末尾保留来源链接。

## 核心概念

用 1-2 段说明项目的主要空间或设计概念。优先写可被图像和空间体验验证的内容，不写空泛形容词。

## 设计策略

## 材料 / 结构 / 建造

记录材料、结构、构造、临时性/永久性、施工方法等信息。没有可靠资料时明确写“公开资料未确认”。
```
