# 钱学森图书馆 / Qian Xuesen Library

## 快速信息

- 项目名称：钱学森图书馆 / Qian Xuesen Library
- 建筑师 / 事务所：公开资料未充分确认；检索结果显示该项目与华南理工大学建筑设计研究院作品列表相关，需以设计单位官网或项目图纸复核
- 地点：中国上海市徐汇区华山路1800号，上海交通大学徐汇校区
- 年份：2011年开馆；建设筹备可追溯至2002年，2010年奠基
- 类型 / 功能：纪念性图书馆、人物纪念馆、专题展陈空间、校史/科学传播空间
- 状态：已建成并开放；2024年入选国家一级博物馆名单
- 主要来源：钱学森图书馆官网入口、维基百科条目、中国新闻网/国家一级博物馆名单相关页面、华南理工大学建筑设计研究院条目线索

## 一句话概括

上海交通大学钱学森图书馆把“校友纪念、国家科技叙事、文献收藏和公众展陈”压缩进一座中小尺度校园纪念建筑中，其研究价值在于纪念性建筑如何通过立面图像、展陈序列和校园位置建立人物叙事。

## 项目背景

钱学森图书馆位于上海交通大学徐汇校区，公开资料记录地址为上海市徐汇区华山路1800号。维基百科条目记载，该馆于2011年12月钱学森100周年诞辰时开馆，总用地面积约8,960平方米，总建筑面积约7,960平方米，陈展面积3,000多平方米，共四层，地上三层、地下一层。[维基百科：钱学森图书馆（上海交通大学）](https://zh.wikipedia.org/wiki/%E9%92%B1%E5%AD%A6%E6%A3%AE%E5%9B%BE%E4%B9%A6%E9%A6%86_%28%E4%B8%8A%E6%B5%B7%E4%BA%A4%E9%80%9A%E5%A4%A7%E5%AD%A6%29)

该馆并不是普通校园图书馆，而是围绕钱学森文献、手稿、书籍、照片和实物展开的纪念性收藏与展示机构。条目记录其馆藏包括钱学森相关文献、手稿和书籍76000余份，珍贵图片1500余张，实物700余件。2024年，第五批国家一级博物馆名单中也出现了“钱学森图书馆（上海交通大学）”，说明它在功能上已被纳入博物馆评价系统。[国家一级博物馆](https://zh.wikipedia.org/wiki/%E5%9B%BD%E5%AE%B6%E4%B8%80%E7%BA%A7%E5%8D%9A%E7%89%A9%E9%A6%86)

## 核心概念

这个案例的核心不是“大体量公共图书馆”，而是“人物纪念馆式图书馆”。建筑外观中可见由瓷砖拼贴形成的钱学森肖像，这使立面直接成为纪念叙事的媒介：建筑不是只容纳展览，而是在城市/校园界面上先行表达“纪念谁”。[维基百科：钱学森图书馆（上海交通大学）](https://zh.wikipedia.org/wiki/%E9%92%B1%E5%AD%A6%E6%A3%AE%E5%9B%BE%E4%B9%A6%E9%A6%86_%28%E4%B8%8A%E6%B5%B7%E4%BA%A4%E9%80%9A%E5%A4%A7%E5%AD%A6%29)

内部展陈以四个专题组织钱学森的生命与贡献：公开资料列出一、二层设置四个展厅，主题分别围绕“中国航天事业奠基人”“科学技术前沿的开拓者”“人民科学家风范”和“战略科学家的成功之道”。因此，它的空间逻辑更接近“按人物贡献展开的叙事序列”，而不是按馆藏类型分类的传统图书馆。

## 设计策略

1. **用立面肖像建立纪念对象**：外墙肖像让建筑在远距离和入口前就完成识别，不需要依赖说明文字才知道其纪念主题；这是纪念馆常见的“符号前置”策略。[维基百科：钱学森图书馆（上海交通大学）](https://zh.wikipedia.org/wiki/%E9%92%B1%E5%AD%A6%E6%A3%AE%E5%9B%BE%E4%B9%A6%E9%A6%86_%28%E4%B8%8A%E6%B5%B7%E4%BA%A4%E9%80%9A%E5%A4%A7%E5%AD%A6%29)
2. **以专题展厅替代单一阅览逻辑**：四个展厅围绕人物身份、科技贡献、精神风范和战略思想展开，使空间动线服务于叙事推进，而不是单纯服务藏书检索。[维基百科：钱学森图书馆（上海交通大学）](https://zh.wikipedia.org/wiki/%E9%92%B1%E5%AD%A6%E6%A3%AE%E5%9B%BE%E4%B9%A6%E9%A6%86_%28%E4%B8%8A%E6%B5%B7%E4%BA%A4%E9%80%9A%E5%A4%A7%E5%AD%A6%29)
3. **把校园记忆与国家叙事叠合**：选址在钱学森母校上海交通大学徐汇校区，使个人求学背景、学校身份和国家航天叙事在同一建筑中重合；这是“高校纪念馆”的关键场地策略。[维基百科：钱学森](https://zh.wikipedia.org/wiki/%E9%92%B1%E5%AD%A6%E6%A3%AE)
4. **以中小尺度承载高密度文献与展陈**：约7,960平方米建筑面积承载文献收藏、图像、实物与四个展厅，形成一种压缩型纪念馆：体量不大，但信息密度高。[维基百科：钱学森图书馆（上海交通大学）](https://zh.wikipedia.org/wiki/%E9%92%B1%E5%AD%A6%E6%A3%AE%E5%9B%BE%E4%B9%A6%E9%A6%86_%28%E4%B8%8A%E6%B5%B7%E4%BA%A4%E9%80%9A%E5%A4%A7%E5%AD%A6%29)

## 空间与体验

空间体验可以理解为从“人物识别”进入“贡献叙事”：外部通过钱学森肖像建立纪念主题，内部通过四个展厅组织参观路径。对建筑学习来说，值得重点观察的是展厅之间是否形成清晰的时间线、主题线或精神线，以及入口、门厅、楼梯/坡道等过渡空间如何处理“图书馆”和“纪念馆”的双重身份。

## 材料 / 结构 / 建造

公开可读资料确认外墙存在瓷砖拼贴肖像这一视觉特征，但未确认主要结构体系、幕墙/外墙构造、室内材料、展陈系统、施工单位和造价。设计单位也需要进一步通过设计院官网、项目图纸或学校建设档案核实。

## 场地关系

项目位于上海交通大学徐汇校区，这一点让它不只是城市中的独立纪念馆，而是嵌入高校历史场地的纪念建筑。钱学森本人为交通大学1934届校友，因此建筑的场地意义来自“人物与母校”的直接关联，而非单纯交通可达性或商业展示价值。

## 可疑或缺失信息

- “钱学森图书馆”存在上海交通大学与西安交通大学两个同名对象；本案例包默认研究上海交通大学徐汇校区馆。
- 设计单位/主创建筑师未在当前可读官方页面中确认；华南理工大学建筑设计研究院条目将“钱学森图书馆（上海交通大学）”列入其作品线索，但仍需官方项目页复核。
- 钱学森图书馆官网和其“建设基本概况”历史页面在当前访问中读取不稳定，部分关键事实来自二手整理页面，后续应优先补充官网或档案馆资料。
- 结构体系、材料系统、室内展陈设计团队、施工单位、造价和详细图纸信息未确认。
