# 白云国际会堂 / Guangzhou Baiyun International Convention Center International Hall

## 快速信息

- 项目名称：白云国际会堂 / 广州白云国际会议中心国际会堂
- 建筑师 / 事务所：公开资料未充分确认
- 地点：中国广东省广州市白云区白云新城，白云大道南1039-1045号一带
- 年份：广州白云国际会议中心一期2007年竣工；二期/国际会堂公开资料显示于2023年2月正式投用
- 类型 / 功能：会议中心、政务会议、会展、宴会、演出与酒店配套综合体
- 状态：已建成并运营
- 主要来源：岭南东方酒店/广州白云国际会议中心运营页面、广州白云国际会议中心条目、白云新城条目、广州日报相关报道线索

## 一句话概括

白云国际会堂是广州白云国际会议中心体系中的扩展型会议设施，其研究价值在于大型政务会议建筑如何通过“会议、展览、酒店、演出、城市新区公共服务”复合在一起，成为白云新城的公共文化节点。

## 项目背景

广州白云国际会议中心位于广州白云新城中心区，东靠白云山、鸣泉居，西临白云新城中心，南临广州体育馆，北临黄石路，原址与东方乐园的城市更新有关。维基百科条目记录，一期起造于2005年10月26日，2007年4月26日竣工；2023年2月17日，广州白云国际会议中心二期项目正式投用。[广州白云国际会议中心](https://zh.wikipedia.org/wiki/%E5%B9%BF%E5%B7%9E%E7%99%BD%E4%BA%91%E5%9B%BD%E9%99%85%E4%BC%9A%E8%AE%AE%E4%B8%AD%E5%BF%83)

运营方页面将广州白云国际会议中心描述为岭南集团旗下成员、ICCA会员单位，集会议、展览、酒店、餐饮、演出等功能于一体，并说明其与白云国际会堂共同构成总规模超过45万平方米的大型会议会展综合体。[广州白云国际会议中心岭南东方酒店](https://www.lnhotels.com/brand_df/hotels/index/hotel_id/6.html)

## 核心概念

这个案例的核心不是单体地标式会堂，而是“会议城市综合体”。一期会议中心已经包含会议厅、展览厅、宴会厅和酒店；白云国际会堂/二期的加入，使它从单一会议中心扩展为更大规模的复合会展节点。对建筑研究来说，它更适合作为“政务会展综合体 + 城市新区公共设施”的案例，而不是只看立面造型。

从城市层面看，白云国际会堂与白云国际会议中心、广州体育馆、广州市城市规划展览中心、广东画院、白云区图书馆新馆等共同构成白云新城的文体艺术设施群。[白云新城](https://zh.wikipedia.org/wiki/%E7%99%BD%E4%BA%91%E6%96%B0%E5%9F%8E)

## 设计策略

1. **以综合体方式承接高强度会议需求**：会议、展览、酒店、餐饮、演出被组织在同一片区，减少大型活动中的跨城转换成本，形成“住宿-会议-宴会-展演”的连续服务链。[广州白云国际会议中心岭南东方酒店](https://www.lnhotels.com/brand_df/hotels/index/hotel_id/6.html)
2. **多尺度会议空间叠合**：运营页面显示中心拥有66间专业会议厅，可承接20-2600人不同规模会议，并配有30000平方米多功能展场；这说明其空间系统不是一个大厅，而是由大会堂、中型会议室、小型会议室、展场和宴会空间组成的会议矩阵。[会议宴会页面](https://www.lnhotels.com/brand_df/meeting/index/hotel_id/6.html)
3. **以新区公共设施群塑造城市节点**：白云国际会堂不是孤立建筑，而是白云新城公共服务组团的一部分，与会议中心、体育馆、规划展览中心等共同强化片区公共性。[白云新城](https://zh.wikipedia.org/wiki/%E7%99%BD%E4%BA%91%E6%96%B0%E5%9F%8E)
4. **以二期扩展回应运营升级**：2023年二期投用使会议中心体系获得更大容量和数字化运维线索，体现大型会展建筑从“建成使用”向“持续扩容和智慧运维”演进。[广州白云国际会议中心](https://zh.wikipedia.org/wiki/%E5%B9%BF%E5%B7%9E%E7%99%BD%E4%BA%91%E5%9B%BD%E9%99%85%E4%BC%9A%E8%AE%AE%E4%B8%AD%E5%BF%83)

## 空间与体验

白云国际会堂应放在白云国际会议中心整体空间系统中理解：大型会议、宴会和展演空间提供主活动场景，中小会议室承担分论坛、洽谈、接见和辅助功能，酒店和餐饮则支持长周期会议接待。运营页面对岭南大会堂、世纪大会堂、珠江厅、国际会议厅等空间的描述显示，其体验重点在“可分可合、多厅并行、政商务接待完整配套”。

## 材料 / 结构 / 建造

公开可读资料暂未确认白云国际会堂的主创建筑师、设计单位、结构体系、幕墙材料、声学顾问、展陈/室内设计团队和施工单位。可确认的是会议中心体系中部分大会堂按音乐厅/会议厅声学和视听设备需求进行设计，包含同声传译、声光系统、电子表决、会场监控等会议技术配置。[会议宴会页面](https://www.lnhotels.com/brand_df/meeting/index/hotel_id/6.html)

## 场地关系

项目位于白云新城，靠近白云山与广州体育馆，属于广州北部城市副中心公共服务设施群的一部分。这个场地关系使它承担两层角色：一是服务省市级政务与商务会议，二是作为白云新城文体艺术设施网络中的大型公共节点。

## 可疑或缺失信息

- 用户输入“白云国际会堂”，公开资料中更常见的完整名称是“广州白云国际会议中心国际会堂”，且常与“广州白云国际会议中心二期”关联；本案例包按该对象处理。
- 独立的白云国际会堂建筑资料不足，部分事实来自广州白云国际会议中心整体资料，不能完全等同于国际会堂单体。
- 主创建筑师、设计单位、结构体系、立面材料、施工单位和准确建筑面积未在当前可读资料中确认。
