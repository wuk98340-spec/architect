# BIG Maze / The BIG Maze

## 快速信息

- 项目名称：BIG Maze / The BIG Maze
- 建筑师 / 事务所：Bjarke Ingels Group (BIG)，Bjarke Ingels
- 地点：National Building Museum, Washington, D.C., United States
- 年份：2014
- 类型 / 功能：临时装置、展览空间、可步入式迷宫
- 状态：临时展出，公开资料显示展期至 2014-09-01
- 主要来源：BIG PIXEL 项目页、Architectural Digest、Wired

## 一句话概括

BIG Maze 把传统迷宫“越走越迷失”的体验反转为“越接近中心越清楚”的空间装置，用墙高变化把迷宫从困惑机器变成可被理解的公共体验。

## 项目背景

BIG Maze 是 Bjarke Ingels Group 为 Washington, D.C. 的 National Building Museum 设计的临时装置，设置在博物馆 Great Hall 内。Architectural Digest 将其描述为 BIG 更大规模展览 “amBIGuity” 的预览项目，并记录其开放日期为 2014-07-04、展期至 2014-09-01。[Architectural Digest](https://www.architecturaldigest.com/story/big-maze-national-building-museum-bjarke-ingels)

该项目延续了欧洲 17-18 世纪园林迷宫的空间原型，但不再把“迷路”作为唯一目的，而是把中心点设计成理解整体路径的观察位置。[Wired](https://www.wired.com/2014/07/a-clever-maze-that-actually-gets-less-confusing-as-you-reach-the-center)

## 核心概念

传统迷宫通常通过等高墙体制造遮挡和不确定性；BIG Maze 则让墙体高度从外围向中心逐渐降低，形成类似倒置圆锥的空间效果。人进入外围时面对接近峡谷般的高墙，越向中心行进，视野越开放，最终能从中心回看自己刚刚走过的路径。

这个反转让迷宫从“失控感”转向“逐步获得清晰度”。它保留了探索和绕行的游戏性，但用可见性、俯瞰视角和中心回望把体验转译为一种空间认知过程。

## 设计策略

1. **反转迷宫逻辑**：把传统迷宫中越深入越复杂的体验，改写为越接近中心越清晰的体验；这是项目最核心的空间策略。[Wired](https://www.wired.com/2014/07/a-clever-maze-that-actually-gets-less-confusing-as-you-reach-the-center)
2. **用墙高组织心理感受**：外围墙体高，制造压迫和不确定；内部墙体降低，逐步释放视野，把焦虑转化为掌控感。[Wired](https://www.wired.com/2014/07/a-clever-maze-that-actually-gets-less-confusing-as-you-reach-the-center)
3. **让博物馆剖面参与观看**：装置位于 Great Hall，观众可从二、三层廊道俯瞰整体，使迷宫同时拥有“身在其中”和“整体观看”两种体验。[Architectural Digest](https://www.architecturaldigest.com/story/big-maze-national-building-museum-bjarke-ingels)
4. **以普通材料制造强烈空间图解**：中性色 maple plywood 强化了迷宫的几何和尺度，而不是依赖装饰性图案。[Architectural Digest](https://www.architecturaldigest.com/story/big-maze-national-building-museum-bjarke-ingels)

## 空间与体验

体验从外围高墙开始，身体首先感知到窄缝、遮挡和迷失；随着路径推进，墙体高度降低，人的视线开始越过墙面获得方位感。中心点成为体验转折：它不是终点纪念碑，而是一个可回看、可理解路径结构的观察点。

## 材料 / 结构 / 建造

公开资料明确提到材料为 neutral-tone maple plywood。Architectural Digest 记录外围墙高约 18 feet；Wired 记录平面尺度约 60 x 60 feet。具体构造节点、承重方式、预制/现场装配方法在当前公开资料中未确认。

## 场地关系

项目的尺度依赖 National Building Museum 的 Great Hall：高大中庭容纳了 18 feet 的外围墙，也允许观众从上层廊道观看整体结构。因此，迷宫不是孤立物件，而是借助博物馆内部剖面形成“进入”和“俯瞰”的双重读法。
